To start :
1-get your self bot token and put it your .env file in this format
	token = 123123
2-in settings.json replace the myself strings with strings of user ids of the accounts you want the bot to respond to their commands
and replace the log channel string with the channel id the bot will send its logging messages to
3-run A_main.py file and test if the bot is running by sending "selfcall test" in a server the bot can read
